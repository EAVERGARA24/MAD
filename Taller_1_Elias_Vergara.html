    plt.figure(figsize=(10, 6))

    # Datos originales
    plt.scatter(X1, X2, c=y, cmap='bwr', marker='o', edgecolor='k', s=50, label='Datos originales')

    # Límite de decisión
    xlim = (X1.min() - 1, X1.max() + 1)
    ylim = (X2.min() - 1, X2.max() + 1)
    xx, yy = np.meshgrid(np.linspace(xlim[0], xlim[1], 500), np.linspace(ylim[0], ylim[1], 500))
    Z = model.decision_function(np.c_[xx.ravel(), yy.ravel()])
    Z = Z.reshape(xx.shape)

    plt.contourf(xx, yy, Z, alpha=0.3, cmap='bwr', levels=[-1, 0, 1])
    plt.colorbar(label='Decisión')
    plt.xlabel('X1')
    plt.ylabel('X2')
    plt.title(title)
    plt.legend()
    plt.show()

Impacto de C en el Modelo (iii)¶

A mayor regularización (0.001): lo que lleva a una frontera de desición mas flexible pero con mayor margen de error en la clasificación de los puntos.

A menor regularización (100): lo que lleva a una frontera de desición mas estricta pero con menor margen de error en la clasificación de los puntos.

Comparación con la regresión Logistica (iv)¶

Regresión Logistica: Tiende a tener fronteras de desición basadas en probabilidades que van a seguir un comportamiento rigido. SVM con C de 1 : Regulariza permitiendo márgenes mas amplios y fronteras de desición más flexibles lo cual permite que se de un mejor un comportamiento mas preciso versus los valores historicos.
